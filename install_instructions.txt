## Installation Instructions

1. **Create the directory structure:**
   ```cmd
   mkdir c:\src\dockergame
   cd c:\src\dockergame
   mkdir shared
   mkdir src

Create all the files as shown above in their respective directories
Start Ollama on your Windows 11 host:
cmdollama serve

cmddocker-compose up --build
